Audio Visualizer Documentation

Audio Effect Nodes

  • Highshelf: This filter is an instance of BiQuadFilter. It changes the loudness of higher frequencies of sound.
  • Lowshelf: Similar to highschelf, this filter changes the loudness of lower frequencies of sound.
  • Distortion: This filter transforms frequencies using a certain formula.

Sprite Class

My sprite class takes in the audio data and uses it to change the width and height. The width represents the lower frequencies, while the height represents the higher frequencies. The draw function uses all of the fields for position, scale, and size, while the update function takes in the new audio data every frame and calculates the scaling. The exact formula used includes taking the average of a range of frequencies. The 8 lowest are counted as bass while the rest are counted as treble.

Asthetics

These are the main things I changed in my app:

  • Radial Display: I changes the previous circles into a radial display. The angle of each segment is reflected by the frequency, while the radius and color are reflective of the loudness.
  • Spectrogram: I replaced the bars with a spectrogram. I created a new class to store a frame of audio. The class draws the frame in segments from the top to the bottom of the screen; low frequency to high frequency. The color and width of each segment reflects the amplitude of loudness of that frequency
  • Gradient: I changed the gradient to be at an angle and have more meloow tones to not be so garish
  • Blur: I implemented Gaussian blur as an option to effect the canvas. Unfortunately since it uses the CPU and is on a browser it becomes very laggy. However, I did complete quite a bit of research and development to understand the formula and procceses.
